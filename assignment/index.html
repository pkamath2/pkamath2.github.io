Pitch estimation for monophonic acoustic guitar notes using gated recurrent neural networks

 
 
 
This is a companion webpage for above titled paper. The code for this project is on Github and can be found here
 
 
 
Examples
Below are 4 pitches from the NSynth validation set with annotated MIDI numbers 24 (f0 = 32.7Hz), 34 (f0 = 58.27Hz), 53 (f0 = 174.61Hz), and 75 (f0 = 622.25Hz). Each example is 4 seconds in length and is sampled at 16kHz.

A 2048 sample is extracted every 128ms and pitch is estimated using Pitch GRU (our algorithm), pYIN and CREPE. Additionally, PyTorch audio's (torchaudio) internal implementation is also used to estimate pitch for comparison.

For lower pitches (below 40) Pitch GRU gives more stable estimates than pYIN and CREPE. Pitched higher than MIDI 40 Pitch GRU, pYIN and CREPE have approximately similar accuracies. Also, towards the end of the signal (around 3 second mark in each example) all algorithms estimate incorrectly, except pYIN which eventually corrects its estimates.
